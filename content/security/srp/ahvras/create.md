漏洞扫描服务只支持包年包月的购买方式。您需要选择合适的服务版本和购买时长，并一次性完成预付费。

## 计费规则

> 购买时长满一年可享受 **8** 折优惠。

| **计费版本** | **价格**   |
| ------------ | ---------- |
| 基础版       | 2500 元/月 |
| 标准版       | 5000 元/月 |
| 高级版       | 7500元/月  |
| 旗舰版       | 10000元/月 |
| 专业版       | 12500元/月 |

## 操作步骤

1. 进入**安全资源池**页面。

2. 在左侧导航栏选择**漏洞扫描**。

   进入**扫描漏洞**页面。

3. 点击**购买扫描漏洞**。

4. 选择版本、购买时长和区域，并勾选**服务协议**。

   | **漏洞扫描版本** | **每个模块支持IP地址数量（个）** |
   | ---------------- | -------------------------------- |
   | 基础版           | 20                               |
   | 标准版           | 50                               |
   | 高级版           | 100                              |
   | 旗舰版           | 200                              |
   | 专业版           | 300                              |

5. 点击**立即购买**并完成支付。
